Metadata Quality Assessment (MQA) Methodology

The MQA is a tool developed to assess the quality of metadata following DCAT standards. It helps data providers and portals evaluate their metadata quality and receive improvement suggestions.

Key Features

  • Download Reports: Get detailed quality assessment reports
  • Dashboard: Visual overview of metadata quality metrics
  • Catalogues: Manage and assess multiple data catalogs
  • Methodology: Based on FAIR principles and DCAT-AP standards

Quality Dimensions

  • Findability (100 pts): Keywords, categories, geo/time search
  • Accessibility (100 pts): URL accessibility, download links
  • Interoperability (110 pts): Format standards, DCAT-AP compliance
  • Reusability (75 pts): Licensing, contact info, publisher data
  • Contextuality (20 pts): Rights, file size, dates

SHACL Validation Levels

  • Level All: Base, vocabularies, recommended properties, ranges
  • Level 3: Base, vocabularies, recommended properties
  • Level 2: Base, vocabularies
  • Level 1: Base only

Quality Rating Scale

Rating Points Range
Excellent 351 - 405
Good 221 - 350
Sufficient 121 - 220
Bad 0 - 120
